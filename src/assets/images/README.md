# Assets Directory

This directory contains image assets for the website.

## Avatar Image

Please replace `avatar-placeholder.png` with the actual avatar image exported from Figma.

The original Figma asset ID was: `e61f386b2c2ac454de982581bf3b2dad6dceffa4`

To export:
1. Open the Figma design file
2. Select the avatar image
3. Export as PNG (recommended size: 128x128 or higher for retina displays)
4. Save it as `avatar.png` in this directory
5. Update the import in `src/App.tsx` if you change the filename
